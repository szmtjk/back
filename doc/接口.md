```

入参：
{
  "name": "港口34",			//港口名
  "portNo": "1234",			//港口编号
  "portType": 1,				//港口类型  1：集装箱 2：其他

  "latitude": 34124534,
  "radius": 1200,
  "description": ""
}

{

  "errCode": 200,
  "msg": null,
  "data": 9

#### 2.1 新增
```
接口名:waterLevel/add

{
    "name":"水位线01"，
    "level":100,
