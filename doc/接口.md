### 0. 约定
列表查询的默认参数
```
{
  "queryParamFlag": 0, //默认为0， 为1时标识mini ui特殊格式
  "page": 1, 
  "pageIndex": 1, //queryParamFlag为1时生效
  "pageSize": 10
}
```
当queryParamFlag为1时，返回格式为：
```
{
  "success": true,
  "msg": "***"
  "total": 10,
  "data": [{...
  }]
}
```

### 1. 港口
#### 1.1 新增
```
接口名:port/add
入参：
{
  "name": "港口34",			//港口名
  "portNo": "1234",			//港口编号
  "portType": 1,				//港口类型  1：集装箱 2：其他
  "company": "xx公司",
  "longitude": 118123567,
  "latitude": 34124534,
  "radius": 1200,
  "description": ""
}

出参：
{
  "success": true,
  "errCode": 200,
  "msg": null,
  "data": 9
}
```
